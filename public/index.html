Settings

Scheduled scans
TreeMap re-scans these folders on a schedule while the app is running, and alerts you when a folder grows past your threshold since the previous scan. Every run is also saved to Trends.
Disk-full forecast
Scheduled scans also project when the disk will fill up (needs 5+ scans across 2+ days). You'll get a notification when the forecast drops below:
days of the disk filling up
Live activity
The treemap's Live mode watches the scanned folder in real time and pauses itself when nothing is happening.
minutes without disk activity
Cloud accounts
Scan your own Google Drive, Dropbox or OneDrive in the same treemap — metadata only, no file contents are ever downloaded, and sign-in tokens never leave this computer. Bring your own (free) API app: create one in the provider's developer console with redirect URI http://127.0.0.1:53682/oauth/callback, then paste its client ID below.
Ignore list
Folders, files or patterns to leave alone. Don't scan skips them entirely (faster scans); Don't suggest keeps them out of Smart Suggestions only. Patterns: a full path, a name like node_modules, or globs like *.iso and ~/projects/**/dist.
